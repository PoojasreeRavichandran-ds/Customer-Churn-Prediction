# Customer-Churn-Prediction
Customer Churn Prediction using Machine Learning and Data Analysis
#Project Overview
This project analyzes customer churn behavior and builds machine learning models to predict whether a customer is likely to leave a telecom service.
## Tasks Completed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Machine Learning Model Building
- Model Evaluation
## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
## Model Performance
- Logistic Regression Accuracy: 82.11%
## Key Insights
- New customers showed higher churn rates.
- High monthly charges were associated with increased churn.
- Long-term contracts reduced churn significantly.
